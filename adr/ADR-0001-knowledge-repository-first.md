# ADR-0001: Knowledge Repository First

## Status

Accepted

## Context

Project Alter Ego could easily become only a website, resume, or branding exercise.

That would be too narrow.

The long-term goal is to create a durable professional operating system for Amit Bhushan Bhatt.

## Decision

Project Alter Ego will be treated as a knowledge repository first.

The website, resume, LinkedIn profile, talks, articles, consulting pages, and future AI-generated artifacts will be outputs generated from this repository.

## Consequences

- Source documents are more important than final artifacts.
- Identity, mission, values, career narrative, and knowledge architecture must be maintained as first-class assets.
- Generated artifacts should not become the source of truth.
- Future AI tools should read this repository before producing artifacts.
